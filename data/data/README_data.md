# Fuente de datos

Este proyecto utiliza un dataset educativo que simula el comportamiento de clientes de una empresa de telecomunicaciones (Megaline).

## Archivos incluidos

- megaline_users.csv → Información demográfica y plan contratado.
- megaline_plans.csv → Detalles de los planes tarifarios.
- megaline_calls.csv → Registros de llamadas por usuario.
- megaline_messages.csv → Registros de SMS por usuario.
- megaline_internet.csv → Consumo de datos móviles por usuario.

## Uso del dataset

1. Los archivos deben mantenerse dentro de la carpeta `/data/`.
2. Ejecutar el notebook ubicado en `/notebooks/`.
3. El análisis consolida consumo mensual y calcula ingresos por usuario.

## Nota

El dataset es educativo y fue utilizado con fines académicos.
